# GymPOO

GymPOO es una aplicación de escritorio desarrollada en Java y JavaFX enfocada en la administración de gimnasios. El proyecto tiene como objetivo practicar programación orientada a objetos, interfaces gráficas, manejo de archivos y arquitectura básica de aplicaciones.

## Características planeadas

- Registro de clientes
- Gestión de membresías
- Control de pagos
- Administración de empleados
- Interfaz gráfica con JavaFX
- Persistencia de datos mediante serialización
- Listas dinámicas usando ArrayList
- Arquitectura basada en Models, Views y Controllers

---

## Tecnologías

- Java
- JavaFX
- Scene Builder
- Maven
- Serialización de objetos
